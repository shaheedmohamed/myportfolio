================================================================================
                            QUICK START
================================================================================

1. INSTALL & RUN
   cd SecureVault
   npm install (already done)
   npm start

2. RUN ON ANDROID
   npm run android

3. RUN ON iOS
   npm run ios

4. FIRST TIME SETUP
   • Launch app
   • Go to Settings tab
   • Enable PIN Protection
   • Enter 4-8 digit PIN
   • Confirm PIN
   • Go to Vault tab
   • Tap "➕ Import File"
   • Select media from device
   • File is encrypted and stored securely

ENCRYPTION
  ✅ Algorithm: AES-256-CBC (military-grade)
  ✅ Key Size: 32 bytes (256-bit)
  ✅ IV Size: 16 bytes (128-bit)
  ✅ Key Storage: Platform-native secure store
  ✅ File Format: .enc extension

AUTHENTICATION
  ✅ PIN Length: 4-8 digits
  ✅ PIN Hashing: SHA-256
  ✅ PIN Storage: Secure store (encrypted)
  ✅ Verification: Hash comparison
  ✅ No Recovery: By design (security feature)

STORAGE SECURITY
  ✅ Location: App-private directory
  ✅ Hidden: .nomedia file prevents indexing
  ✅ Permissions: App-only access
  ✅ Original Files: Automatically deleted
  ✅ Metadata: Stored with encrypted files
